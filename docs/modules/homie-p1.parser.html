P1Parser:push_line (line)
Line-by-line parser function. Lines must be fed in line by line. Lines will be buffered until the datagram is complete.

Parameters:

  • line string read from the P1 port

Returns:

    table parsed datagram or nil+error. If the error is "incomplete" then more lines are needed to complete the datagram.
